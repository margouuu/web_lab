Перелік тегів, атрибутів та правил CSS

  1. < a >-Створює посилання
  2. < frame >-Визначає фрейм на веб-сторінці
  3. < frameset >-Визначає структуру фреймів
  4. < iframe >-Вставляє на сторінку фрейм
  5. < img >-Відображає графічне зображення на веб-сторінці
  6. < ol >-Визначає впорядкований список.
  7. src="URL"-Вказує адресу файлу (URL), який буде завантажуватися у фрейм
  8. < Style >-включає таблиці стилів
  9. href="URL"-Задає адресу документа, на який слід перейти
